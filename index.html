One-time setup

CineWatch pulls US theatrical release dates from TMDB, a free movie database, as an estimate for when a movie is likely to hit Ecuador — TMDB's Ecuador-specific data is too sparse to browse on its own. Get a free API key, then paste it here.

1. Create a free account at themoviedb.org
2. Go to Settings → API → Request an API key (choose "Personal")
3. Paste your API Read Access Token below

Install CineWatch to your Home Screen to get real notifications on this iPhone — tap Share → Add to Home Screen.

Turn on notifications so CineWatch can tell you the moment a movie releases or hits cheap-day pricing.
